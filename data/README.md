# Automated Reporting Pipeline

AI-powered reporting tool for Weather, Stocks and E-commerce data.

## Features
- Live weather forecasts with charts
- Stock market analysis with candlestick charts
- E-commerce CSV/Excel upload and analysis
- AI-generated executive summaries
- PDF and TXT report download
- Auto-scheduled daily reports

## How to Run
pip install -r requirements.txt
streamlit run app.py

## API Keys needed
- GROQ_API_KEY — free at console.groq.com
- ALPHA_VANTAGE_KEY — free at alphavantage.co
